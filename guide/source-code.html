# async placeOrder(currency, network, addresses, amounts, hashes) -> object

Calls the sendBulkTransaction method of the Escrow Bulk Transactions smart contract with the current MetaMask account.
Returns a TransactionResponse which includes all properties of a transaction.

Param Type Description
currency string Name of the blockchain in order to determine the correct currency
network string Network type of blockchain - mainnet or testnet
addresses array Blockchain addresses of the shops which will potentially receive tokens
amounts array Total price of the purchased product/products for every shop
hashes array Generated hash of the every purchased item

Addresses length and amounts length must be equal.

source (opens new window)

# async addShopToContact(contact, { shop, status }, location) -> void

Adds an object ({ shop, status }) to contact according to given location in contacts.json file. Location can be either "myShops" or "theirShops". Someone else has a read access to the current user's shop/shops in "myShops". The current user has a read access to someone else's shop/shops in "theirShops".

Param Type Description
contact string A Solid protocol WebId used as an object key
{ shop, status } object Object with shop name and the status fetched from RequestingAccess modal
location string Either "myShops" or "theirShops"

source (opens new window)

# async setShopAccessForAgent(agent, shopName, rulesObj) -> void

Creates an acl files for different WebId than the current user with read access. After the function finishes the other user has an accces to:

  • Product Images Solid Dataset for a given shop
  • Shops Solid Dataset for a given shop
  • Shop Covers Solid Dataset for a given shop
  • Shop Details Solid Dataset for a given shop
Param Type Description
agent string A Solid protocol WebId used as an object key
shopName string The name of a shop
rulesObj object Object containing different accesses. For this function only has an read access
